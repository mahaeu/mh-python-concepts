# A little introduction to raw SQL

to use the playground notebooks (.ipynb files)
1. Open init_db.py and run it by clicking the "Run" button in VS Code.
   This will create a SQLite database file named `docs.db` in the same directory.
2. Open any of the playground notebooks and run the cells to interact with the database.
3. try out export_and_use.ipynb to see how to export data from the database to .csv and use it in pandas.
